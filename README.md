# 2IMV20 Visualization and Interaction - 2020 - Jolan Rensen & Daan Waalboer

You can build and run the project by executing:

    gradlew browserRun
or on Windows

    gradlew.bat browserRun


Alternatively, you can build the project with:

    gradlew(.bat) build
And then you can open 

    ./build/distributions/index.html
to view the website.
